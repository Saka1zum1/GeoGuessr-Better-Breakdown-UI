### Manual Installation

1. Click the Tampermonkey icon in your browser
2. Select "Create a new script"
3. Copy the contents of `GeoGuessr Better Breakdown UI.js` and paste it into the editor
4. Press `Ctrl+S` (or `Cmd+S` on Mac) to save
